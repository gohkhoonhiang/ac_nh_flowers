 Help mdi-help What is this site about?

Since I've already created my own AC:NH app for critters, villagers and more, I might as well also put the flowers data in the app so I can view them all in one app.

 Where do you get your data from?

As mentioned in the previous answer, I got my data from the Fandom flowers list.

Since the types of flowers are limited, I didn't automate the data formatting. I just created a simple JSON file containing the different flower types and colors.

Additionally, I also make use of the extensive flower genetics guide from an awesome source to create a simple genetics table for breeding purposes. The breeding mechanism is more advanced than what the table can describe, but I would love to have a quick table view for reference when I'm breeding the flowers. If you're interested in how it actually works, I encourage you to go to the source document to read the detailed technical explanation.

 Which browsers are supported?

This site has only been tested on Google Chrome (Version 81.0.4044.129). No guarantee that it will work on other browsers.

 Where did you get the color scheme from?

The color scheme is inspired by the Turnip Prophet site. It's a really useful site I personally use to track my turnip prices and predict when is the best time I should sell. Go and support them!
